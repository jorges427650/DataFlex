# FutbolWebApp

Entrega de la practica DataFlex + MySQL.

Incluye:

- Workspace DataFlex `FutbolWebApp`.
- Script SQL original en `SQL/futbolBD.sql`.
- Conexion a MySQL mediante `DFCONNID=Futbol`.
- Tablas MySQL traidas a DataFlex mediante `.int` y `.fd`.
- Vistas Select y Zoom para `entrenador`, `jugador`, `posicion`, `equipo` y `liga`.
- Tiles en `Dashboard.wo` para las cinco tablas pedidas.

Antes de ejecutar en otro equipo, cambiar la contrasena en:

`FutbolWebApp/Data/DFConnId.ini`

Proyecto verificado localmente: `FutbolWebApp.src` compila sin errores.
